
## Project Overview

SMART LENDER is a Machine Learning-based web application that predicts whether a loan application is likely to be approved or rejected based on the applicant's details. The application helps financial institutions make quick, accurate, and data-driven loan approval decisions.




## Features

- User-friendly web interface
- Loan eligibility prediction using Machine Learning
- Fast and accurate prediction
- Flask-based web application
- Easy-to-use loan application form

## Installation

1. Clone the repository

```
git clone https://github.com/shaiksumaya1792/SMART-LENDER-PROJECT-NEW.git
```

2. Install the required packages

```
pip install -r requirements.txt
```

3. Run the application

```
python app.py
```

- Gender
- Marital Status
- Dependents
- Education

- Applicant Income
- Co-applicant Income
